# KiiKii 기획 위원회 검토 — 2026-05-06

> 논의 주제: 연속출석 / 응원 지표 / 응모권 히스토리 / 전체 보강

## 1. 시스템 기획자 (Ronin)

**발견:**
- 기획서 3-2에 "연속 출석 콤보: N일 연속 응원 시 보너스 응모권" 명시되어 있으나 미구현 — 고아 시스템
- 응모권 소스: 응원(+1) / 카드(+1) 두 개뿐. 광장 글쓰기(+1)도 미구현
- streak 카운터 존재하지만 보상 로직 없어 숫자를 쌓을 이유 없음

**우려:**
- 연속 출석 보너스 없이는 streak이 장식 → D7 리텐션 목표(50%) 달성 근거 약함
- 응모권 경제가 무료 루프 2개에만 의존

**제안:**
- 연속 출석 마일스톤: 3일(+1) / 7일(+3) / 30일(+5) 구간, 출석 캘린더 시각화
- 광장 글쓰기 응모권은 스팸 방지 이슈로 MVP 제외, 출석 마일스톤으로 대체

---

## 2. 콘텐츠 기획자 (Noel)

**발견:**
- 카드북이 기획서 핵심 콘텐츠(100장)인데 현재 진입 경로 없음
- 탭 네비게이션이 기획서(홈|카드|광장|이벤트|프로필)와 현재 구현(홈|광장|이벤트|미니룸|MY) 다름 — 카드 탭 소실

**우려:**
- 카드 수집이 핵심 재미인데 카드북 접근 막히면 수집 동기 사라짐
- 연속 출석 의미 없으면 일 1회 접속 습관 형성 실패

**제안:**
- 출석 캘린더: 이번 달 달력 뷰, 출석일 하트 표시, 마일스톤 날 별 표시
- 응모권 히스토리는 별도 페이지 대신 MY 내 접기/펼치기 인라인으로 충분
- 카드북을 하단 탭에 직접 배치 고려 (기획서 원안 복원)

---

## 3. BM 전문가 (Harry)

**발견:**
- 현재 무료 루프만 돌아가고 유료 루프(데코 구매 → 응모권 보너스) 전혀 없음
- 연속 출석 보상 없으면 D1→D7 전환율 낮아져 과금 노출 기회 감소

**우려:**
- 응모권 히스토리 페이지: 많이 응모했는데 당첨 없으면 허탈감 → 이탈 가속 위험
- 아이템 상점 미구현 → 목업 시연 시 수익 흐름 설명 불가

**제안:**
- "이번 달 내가 모은 응모권 N개" 숫자 강조 + 이벤트 잔여 기간 CTA가 더 효과적
- 연속 출석 마일스톤 보상은 D7 리텐션 직접 기여 — 목업에서도 "7일 개근 +3" 표시 자체가 훅
- 목업 완성도를 위해 아이템 상점 더미 화면 1장 추가 권장

## 4. UX 기획자 (Kay)

**발견:**
- 응원 지표: 기획서에 "개인 기록만, 경쟁 없음" 명시 → 올바른 방향
- 응원 누적 숫자 노출은 자기만족 피드백으로 무방, 경쟁 요소로 변질만 금지

**우려:**
- 홈의 "연속출석 N일" 숫자가 맥락 없이 부유 → 유저가 중요성 모름
- 응모권 히스토리 전용 페이지는 화면 추가 비용 대비 효용 낮음

**제안:**
- 응원 누적: "총 N회 응원했어요 💜" 한 줄 텍스트로 감성 표시
- 연속 출석: 홈 streak 숫자 탭 → 이번 달 출석 캘린더 팝업으로 맥락 제공
- 응모권 히스토리: MY 내 응모 현황에 "획득 내역" 탭으로 통합
