## 상세 결과

### 검증 1: 코어 루프 완결성

- **판정:** ✅ 통과
- **내용:** 루프가 완결 순환함
  ```
  재접속 → 응원+카드 → 응모권 축적 → 이벤트 응모 → 당첨 기대 → 재접속
  ```
  카드북 빈 슬롯, 팬 섬 꾸미기, 광장 채팅이 보조 재방문 동기로 작동.
- **잠재 리스크 (low):** 이벤트 당첨 확률이 낮거나 이벤트 공백이 길어지면 응모권 축적의 의미가 퇴색될 수 있음. 이벤트가 항상 열려 있어야 루프가 끊기지 않음. 상시 트랙으로 설계된 것이 이를 해소하나, 상시 트랙 상품의 매력도 유지가 운영 과제.

---

### 검증 2: 경제 밸런스

- **판정:** ⚠️ 경고
- **severity:** medium
- **내용:** 응모권 소스/싱크 구조는 존재하나 상한이 없음.
  - 소스: 하루 최대 3개(무료) + 데코 구매 보너스(무제한)
  - 싱크: 이벤트 응모 소비
  - 문제: 데코를 대량 구매한 유저가 응모권을 수백 개 축적 후 스페셜 이벤트에 몰아 사용하면 당첨 확률 독점 가능
- **근거:** "과금 경쟁 금지" 원칙과 충돌. 고과금 유저가 이벤트를 독점하면 무과금 팬의 이탈 유발.
- **제안:** 이벤트별 1인당 최대 응모 횟수 상한 설정 (예: 스페셜 트랙 최대 10회). 상시 트랙은 상한 없어도 무방(상품 가치가 낮으므로).

---

### 검증 3: 시스템 간 의존성

- **판정:** ✅ 통과
- **내용:** 모든 시스템이 응모권 경제로 수렴하며 고아 시스템 없음.
  ```
  응원 ──────────────────────┐
  Today 카드 ─────────────────┼──→ 응모권 → 이벤트 응모
  광장 글쓰기(후보) ───────────┤
  팬 섬 데코 구매 보너스 ───────┘
  ```
  오시 선택이 홈/섬 양쪽에 개인화를 제공하며 중심축 역할.
- **개선 가능 포인트 (low):** 카드북과 팬 섬 간 연결 없음. 예: "카드 N장 수집 달성 시 한정 데코 언락" 구조를 추후 추가하면 두 시스템이 시너지 발생. MVP에서는 필수 아님.

---

### 검증 4: 스코프 현실성

- **판정:** ⚠️ 경고
- **severity:** high
- **내용:** MVP 시스템이 7개로 권장(3-5개) 초과. 백엔드 의존성이 다수.
  ```
  복잡도 높은 구현 항목:
  - LINE LIFF SDK 연동 + LINE Login
  - Supabase 스키마 + RLS 설계
  - 가챠 확률 로직 (서버사이드 필수)
  - 아이소메트릭 엔진 → React 컴포넌트 포팅
  - 광장 채팅 실시간 동기화
  - LINE Pay 결제 연동
  - LINE Messaging API 알림
  ```
- **근거:** 1인 순수 개발 기준 2주 초과 확실. 단, AI 코딩 지원(Wilson+Jamie)과 아이소메트릭 엔진 기존 구현 완료 상태를 감안하면 4~6주 범위로 현실적.
- **제안:** LINE Pay, LINE Messaging API를 MVP에서 제외하고 2단계로 이동하는 것을 검토. 결제 없이도 무료 응모권 루프로 서비스 검증 가능. 추후 BM 붙이는 방식.

---

### 검증 5: 아트 디렉션 정합성

- **판정:** ⚠️ 경고
- **severity:** medium
- **내용:** 예상 에셋 총량이 기준(50개)을 크게 초과.
  ```
  카드:          100장 (5멤버 × 20장) — 신규 생성
  카드 프레임:     3종
  chibi 홈용:     5장 (기존 보유)
  chibi 섬용:     5장 — 신규 생성
  이모티콘:       30종 (6종 × 5멤버) — 신규 생성
  UI 아이콘:      5종
  아이소메트릭 타일: 3종 (기존)
  아이소메트릭 데코: 20종 — 신규 생성 (LINE Friends 교체)
  이벤트 배너:    2종
  ─────────────────────────────
  총합: 약 173종
  ```
- **근거:** API 키 5개 로테이션과 Gemini 파이프라인이 확보돼 있어 기술적으로는 가능. 비용 추정 약 $15~25 (kiikiii_20260430.md 교훈 기반: 268개 약 $10.45 비율 참조).
- **제안 1:** 카드를 멤버당 20장 → 10장으로 줄여 론칭. 수요 확인 후 확장. 총량 50장 감소.
- **제안 2:** 이모티콘은 MVP 제외, 2단계로 이동. 30종 감소.
- **제안 3 (의도 확인):** 카드 100장 전량 생성이 확정 방향인가, 아니면 시즌별 순차 공개인가? 시즌 1에 30~40장만 공개하면 비용과 운영 부담 모두 낮아짐.
