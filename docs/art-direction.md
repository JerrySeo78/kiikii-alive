## 에셋 생성 전략

**스타일 앵커:** 기존 chibi 일러스트 5장 (멤버별 1장) — 이미 보유, 모든 카드 생성의 기준

### 생성 순서

1. **기존 chibi 5장 스타일 검수** (Kenny → 스타일 일관성 확인)
2. **카드 프레임 3종** Normal / Rare / Special (Laura 담당)
3. **멤버별 카드 20장 × 5명** 포즈/의상/배경 변형 (Kim 담당, Gemini 파이프라인)
4. **UI 컴포넌트** 탭 아이콘 5종, 버튼, 응모권 배지 (Laura 담당)
5. **이모티콘 세트** 6종 × 5멤버 (Kim 담당)
6. **이벤트 배너** 상시/스페셜 각 1종 (Bashy 담당)

### 에셋 생성 도구

- Gemini 이미지 API (`gemini-2.5-flash-image`)
- API 키 5개 로테이션 적용 (.env.local 설정 완료)
- 429/RESOURCE_EXHAUSTED 발생 시 자동 키 전환
